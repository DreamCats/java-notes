# 查看反编译源码

## 例子
JavaBean.java
```java
public class JavaBean {
    int value = 3;
    final int value2 = 5; 
    static int value3 = 8;
    static int value4 = -1;
    String s = "买";
    
    void getValue() {
        int temp = 4;
    }
}
```
- 终端输入`javac JavaBean.java`得到JavaBean.class文件
- 终端继续输入`javap -c JavaBean > JavaBean.txt`
